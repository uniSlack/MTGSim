Created in VSCode. Make sure you have node.js installed.

First Terminal in home:

    npm start

Second Terminal:

    cd client

    if first time running this to get dependencies: npm install 

    npm start

Backend:    http://localhost:3001/api

frontend:   http://localhost:3000


If security issues bc reinstall, put in each terminal (WARNING, might be security issue, exercise caution):

    Set-ExecutionPolicy -ExecutionPolicy Unrestricted -Scope Process
